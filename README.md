🍽️ Zomato Data Analyst Project

End-to-End Business Intelligence & Data Analytics Solution

This project focuses on analyzing restaurant data from Zomato to generate actionable business insights using SQL, Excel, Power BI, and Tableau. The goal is to demonstrate strong skills in data cleaning, data modeling, KPI development, and dashboard creation.



🚀 Project Objective

The objective of this project is to perform an end-to-end data analysis workflow, starting from raw data to interactive dashboards. This includes:

✔ Data Cleaning
✔ Data Modeling
✔ KPI Development
✔ Business Insights
✔ Visualization & Dashboarding
✔ Decision Support

🛠️ Tools & Technologies Used

SQL – Data extraction, transformation, and querying

Excel – Data cleaning, modeling, and analysis

Power BI – Data modeling, DAX, and interactive dashboards

Tableau – Visualization and storytelling

GitHub – Project documentation and version control

📊 Project Workflow
🔹 Step 1: Data Understanding & Cleaning

Imported raw datasets into SQL and Excel

Handled missing values and data inconsistencies

Standardized formats and removed duplicates

Converted categorical and currency data for consistency

🔹 Step 2: Data Modeling

Created relationships between different sheets

Built a structured data model to support reporting and analysis

Optimized model for performance and scalability

🔹 Step 3: Calendar Table Creation

A dynamic Calendar Table was built using the Datekey_Opening column (minimum to maximum date range).

Columns Created:

Year

Month Number

Month Full Name

Quarter (Q1, Q2, Q3, Q4)

Year-Month (YYYY-MMM)

Weekday Number

Weekday Name

Financial Month (April = FM1 … March = FM12)

Financial Quarter (FQ1, FQ2, FQ3, FQ4)

This helped in:
✔ Time intelligence
✔ Trend analysis
✔ Financial reporting

🔹 Step 4: Currency Conversion

Converted restaurant average cost data into USD

Standardized currency for global comparison

Improved consistency for business decision-making
